# Automation — Módulo 10: Reportes

## Casos a automatizar con Playwright

- Registrar ventas → verificar que el reporte las refleja con totales correctos
- Filtrar por Sucursal A → verificar que solo aparecen ventas de esa sucursal
- Registrar venta con descuento → verificar que el desglose (bruto / descuento / cobrado) es correcto
