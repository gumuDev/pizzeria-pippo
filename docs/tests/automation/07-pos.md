# Automation — Módulo 7: Punto de Venta (POS)

## Casos a automatizar con Playwright

- Venta simple sin promo → verificar total correcto en el resumen
- Venta con promo 2x1 → verificar que el descuento se aplica automáticamente
- Venta con promo 2x1 → verificar que el stock descontado corresponde a qty_physical (x2, no x1)
- Confirmar venta → verificar que la orden aparece en "Pedidos del Día"
