# Automation — Módulo 4: Productos

## Casos a automatizar con Playwright

- Crear producto completo (3 pasos: datos + variantes + receta) → verificar que aparece en el POS
- Desactivar producto → verificar que no aparece en el catálogo del POS
- Configurar precio distinto por sucursal → verificar precio correcto en el POS de cada sucursal
